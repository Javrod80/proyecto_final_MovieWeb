(constant) AuthProvider

Proveedor que maneja el estado de autenticación y proporciona las funciones de login y logout. El proveedor guarda el estado de la autenticación en `localStorage`, permitiendo que persista entre las sesiones. Proporciona funciones para iniciar y cerrar sesión, y para recuperar el estado de autenticación cuando el componente se monta.
